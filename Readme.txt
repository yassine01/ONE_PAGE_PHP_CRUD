#DOC
->'biblio.sql'
une petite BD pour gérer les infos des livres et les utilisateurs.

-> 'index.php'
Une interface pour s'identifier.

-> 'home.php'
La page principale permettant la gestion des informations de livres.
(consultation, ajout, modification, suppression et l'exportation vers un fichier csv)

-> 'server.php'
Elle contient: 
- la connexion avec la BD
- toutes les requetes SQL pour gérer les livres.

-> 'style.css'
la feuille de style pour 'index.php' et 'home.php'

-> 'Livres.csv'
Un exemple pour l'exportation des infos.
